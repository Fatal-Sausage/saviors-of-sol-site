== README

To get up and running:

Clone this repository
```git clone <this repo>```

Navigate to the project root and run
```bundle install```
to install required gems

Start the rails server with
```rails s```

And go to 127.0.0.1:3000 in a browser

* Ruby version 2.2.0

* There is currently no database set up, so don't worry about that for now. If we do implement one, it will be a MySQL2 DB
